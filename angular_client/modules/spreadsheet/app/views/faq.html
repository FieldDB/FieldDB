Should I use LingSync for my Fields Methods classes?

No, we do not recommend using LingSync for upcoming fields methods classes. We are planning on discontinuing support for this app on August 1st 2017, after which time corpora will be availible as archive only. This project remains open source and non-proprietary, anyone can use the code to create future apps.

Between February 2015 and November 2015 classes experienced a great deal of student frustration. The current hosting provider and server is unreliable 80%-90% uptime which makes it almost impossible to rely on the app. On December 2nd we upgraded the server, since then it appears to have become more reliable. We expect to change hosting providers after the Christmas break.

If you were planning on using the app, please email us for instructions on how to use Dative.
