  • Offline

  • Offline: Fieldlinguists rarely have internet
  • Semi-structured: Fieldlinguist are super structured, but never the same structure
  • Web widgets: Fieldlinguist want to share their data, but not all of it
  • Encrypted: Fieldlinguist's are bound to laws on the protection and exploitation of 'human subjects'
  • Data intensive: Fieldlinguists would import the web, if they could
  • CPU intensive: Fieldlinguists have lots of repetitive tasks and Opensource projects which automate them.
