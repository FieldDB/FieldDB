This is a place holder for the online version of iField, and a place where you can share a public URL for your corpora (as required by EMELD language data best practices).

The online version of iField works, but only on browsers that have IndexDB working (ie. Chrome and Firefox). We turned off the online version for now until IrishCouch (the company that is providing hosting for the corpora) finishes the CORS module so you can sync your corpora. They expected to finish it in June (before our beta testing) but they got backed up.

With or without CORS, the public URLs will work before December, in Milestone - Field Methods Course Beta Testing

Want to use iField? You can already use iField beta. Go to the website, there are links there that explain what it is (and the link to use it offline).
